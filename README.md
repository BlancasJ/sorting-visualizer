# Sorting Visualizer

A React web app that visualizes sorting algorithms step-by-step with animated bar charts. Supports adjustable array size and animation speed.

## Tech Stack
- React 18
- JavaScript
- CSS

## How to Run
```bash
npm install
npm start
```
Opens at `http://localhost:3000`.

## Features
- Visualizes 5 algorithms: Bubble Sort, Insertion Sort, Merge Sort, Quick Sort, Selection Sort
- Adjustable array size (1-100 elements)
- Adjustable animation speed
- Color-coded bars showing active comparisons and sorted elements
- Generate new random arrays on demand
